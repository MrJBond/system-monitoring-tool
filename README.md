# Linux system monitoring tool

An app written in C++. Provides info about the system, including statistical analysis, building plots in console of the value of multiple OS Linux properties (RAM, processes etc).

Used files
/etc/issue
/proc/version
/proc/diskstats
/proc/meminfo
/proc/cpuinfo
/proc/stat
/proc/vmstat
/proc/uptime

Uses C++ and boost library
